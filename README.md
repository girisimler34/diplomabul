# Diploma Listing Application

## Kullanım Talimatları
Bu uygulama, diploma ilanlarınızı kolayca listelemenizi sağlar. İşte uygulamanın temel özellikleri ve kullanım talimatları:

### Özellikler
- Diploma ilanlarınızı kolayca ekleyin.
- İlanları düzenleyin veya silin.
- İlanlar arasında arama yapın.
- GitHub Pages ile uygulamanızı yayınlayın.

### İlan Ekleme
1. Ana sayfada 'Yeni İlan Ekle' butonuna tıklayın.
2. Zorunlu alanları doldurun ve 'Kaydet' butonuna basın.
3. İlanınız başarıyla eklenecektir.

### İlanları Düzenleme/Silme
1. Listelenen ilanların yanındaki 'Düzenle' veya 'Sil' butonuna tıklayın.
2. Düzenlemek isterseniz gerekli değişiklikleri yapın, ardından 'Güncelle' butonuna basın.
3. Silmek için 'Sil' butonuna tıklayın, ardından onaylayın.

### Arama Fonksiyonu
- Ana sayfadaki arama çubuğunu kullanarak ilanları kolayca arayabilirsiniz.

### GitHub Pages Ayarı
1. GitHub repository'nizde 'Settings' sekmesine gidin.
2. 'Pages' bölümünü bulun ve ana dalınızı seçin.
3. 'Save' butonuna tıklayın. Uygulamanız artık GitHub Pages üzerinden erişilebilir!
